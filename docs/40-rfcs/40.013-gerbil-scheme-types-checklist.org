* Purpose

This checklist tracks the remaining work required for =gerbil-scheme-rust= to
become a general-purpose Gerbil Scheme Rust bindings library, rather than a
Marlin-specific extraction.

The checklist is intentionally type-first. Marlin integration should consume
this crate only after the native Scheme value surface, ownership rules,
scenario coverage, and performance receipts are clear enough to be reused.

* Current evidence baseline

Current Rust ABI evidence:

- =gerbil-scheme-sys= exposes =GerbilStatus=, =GerbilBorrowedUtf8=,
  =GerbilRuntimeOpaque=, =GerbilValueHandle=, and =GerbilI64Callback=.
- =gerbil-scheme= exposes safe =GerbilRuntime=, =GerbilUtf8=,
  =GerbilRuntimeReceipt=, =GerbilValue=, =GerbilI64Callback=,
  =GerbilI64CallbackAbi=, and =NativeError=.
- Scalar native operations cover i64 identity/addition, i64 parity, and i64
  comparison; integer/bytevector conversions additionally cover machine =u64=
  and =i64= with explicit byte order, width, and truncation policy.
- =native-value-surface= scenario covers safe UTF-8, opaque value handles, and
  callback trampoline behavior.
- =source_surface= integration tests cover the Scheme ASP source-surface export
  set, the tracked =scheme/native.ssi= contract, and the rule that
  =scheme/asp/native-surface.ss= stays out of the runtime build.

Current Scheme source-surface evidence:

- =scheme/native.ss= remains the runtime implementation owner.
- =scheme/native.ssi= is the tracked Gerbil signature contract for exported
  bridge symbols.
- =scheme/asp/native-surface.ss= is still the temporary ASP-projected
  workaround for the source-surface view; it must stay out of =build.ss= and can
  be retired once the provider projects =scheme/native.ss= plus
  =scheme/native.ssi= directly.
- The current projected selector set is exactly 27 selectors:
  =gerbil_scheme_rust_abi_version=, =gerbil_scheme_rust_runtime_init=,

  =gerbil_scheme_rust_fixnum_shape=,
  =gerbil_scheme_rust_exact_integer_shape=,
  =gerbil_scheme_rust_char_shape=,
  =gerbil_scheme_rust_flonum_shape=,
  =gerbil_scheme_rust_bytevector_shape=,

* Public backed type matrix

This matrix is the current release-auditable v0 binding surface. A type family
is accepted only when the raw ABI, safe Rust surface, Scheme native-surface
selector, ownership/nullability rule, failure policy, and scenario coverage are
all present. The Rust test =public_backed_type_matrix_covers_current_native_surface=
keeps this table aligned with the Scheme source-surface projection.

| Family | Scheme selector | Raw ABI | Safe Rust surface | Ownership/rooting | Nullability | Failure policy | Scenario |
| status | =gerbil_scheme_rust_status_shape= | =GerbilStatus= | =NativeError::status= | copy status code | not pointer-backed | unknown status preserves code | =status-contract= |
| runtime lifecycle | =gerbil_scheme_rust_runtime_handle_shape= | process-global native runtime init/cleanup | =GerbilRuntime= and =GerbilRuntimeReceipt= | one-shot process-global owner, thread-affine safe handle | not pointer-backed | live re-entry returns =AlreadyInitialized=; finalized runtime returns =RuntimeFinalized=; wrong-thread is blocked by =!Send=/=!Sync= and defensively checked | =native-runtime-round-trip= |
| i64 | =gerbil_scheme_rust_i64_shape= | =i64= | =GerbilRuntime::{identity_i64,add_i64}= | copy scalar | not pointer-backed | overflow maps to =IntegerOverflow= | =native-identity-round-trip= |
| bool | =gerbil_scheme_rust_bool_shape= | =GerbilStatus= plus Rust =bool= return plus Scheme boolean handle | =GerbilRuntime::is_even_i64= and =GerbilValue::as_boolean= | copy scalar or runtime-borrowed Scheme object | not pointer-backed | status fail-closed before bool projection; raw provenance rejected | =native-runtime-round-trip= |
| comparison | =gerbil_scheme_rust_comparison_shape= | =i32= comparison result | =GerbilRuntime::compare_i64= | copy scalar | not pointer-backed | invalid result maps to =InvalidComparisonResult= | =invalid-comparison-fail-closed= |

| fixnum | =gerbil_scheme_rust_fixnum_shape= | =GerbilFixnum= | =GerbilValue::{is_fixnum,as_fixnum,as_fixnum_i64}= | runtime-borrowed Scheme object | not pointer-backed | non-fixnum and raw provenance fail closed | =native-runtime-round-trip= |

| char | =gerbil_scheme_rust_char_shape= | =GerbilChar= | =GerbilValue::{is_char,as_char}= | runtime-borrowed Scheme object | not pointer-backed | invalid Unicode scalar and raw provenance fail closed | =native-runtime-round-trip= |

| bytevector | =gerbil_scheme_rust_bytevector_shape= | =GerbilValueHandle= | =GerbilValue::as_bytevector= and =SchemeBytevector= | runtime-borrowed Scheme object | non-zero handle, distinct from null pointer | non-bytevector objects and raw provenance fail closed | =native-runtime-round-trip= |
| rooted bytes | =gerbil_scheme_rust_rooted_bytes_shape= | =GerbilRootId= | =RootedSchemeString= and =RootedSchemeBytevector= | single-owner Scheme root released by Rust =Drop= | positive root token, zero is conversion failure | invalid input and stale roots fail closed | =bytevector-bytestring-round-trip= |

* Type surface checklist

** Runtime and lifecycle

- [X] Process-global runtime safe owner: =GerbilRuntime=.
- [X] Opaque value handles carry explicit provenance:
  =GerbilValueProvenance::UntrustedRaw= for caller-supplied non-zero raw
  handles.
- [X] Real Scheme object export handles carry explicit provenance:
  =GerbilValueProvenance::SchemeObjectExport= for borrowed/unrooted values
  produced by the initialized Gerbil native module.
- [X] Runtime-sentinel opaque value provenance is distinct from =UntrustedRaw=.
- [ ] Runtime/export-produced opaque value provenance that can safely enable
  Gerbil/Gambit predicates and traversal.
- [X] Thread-affinity check for runtime methods.
- [X] ABI version check before exposing runtime handle.
- [X] Drop-time native cleanup path.
- [X] Explicit runtime re-entry policy document: what is legal after cleanup,
  what is process-final, and what must fail closed.
- [X] Scenario for live re-entry and runtime-finalized behavior.
- [ ] Compile-time or defensive scenario for wrong-thread access. Safe
  =GerbilRuntime= is intentionally =!Send=/=!Sync=, so a runtime wrong-thread
  scenario must not weaken the public API just to make the branch reachable.
- [X] Public receipt type for runtime initialization, ABI id, ABI version, and
  native module path.

** Status, error, and result

- [X] Stable raw status enum: =GerbilStatus=.
- [X] Safe native error enum: =NativeError=.
- [X] Unknown status preservation in status contract tests.
- [X] Panic containment for i64 callback trampoline.
- [ ] Public =GerbilResult<T>= alias or typed result wrapper.
- [?] Scheme-side result shape projection for success/error payloads is present
  as ASP source-surface taxonomy, but no public =GerbilResult<T>= wrapper exists
  yet.
- [ ] Error mapping table from Gerbil exception classes to stable Rust errors.
- [X] Rust scenario/test receipt that validates status/error/result projection
  together, including unknown status preservation and optional status
